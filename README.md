# # Bluetooth Gesture-Controlled Car（STM32+FreeRTOS）
蓝牙姿态控制小车（嵌入式综合实践项目，2026.04-2026.05）：基于 STM32F103C8T6 与 FreeRTOS 搭建蓝牙控制小车系统，采用 MPU6050 姿态传感器、编码器、TB6612 电机驱动与 OLED 显示模块，实现串口蓝牙指令控制小车前进、后退、左/右转指定角度及按设定速度运行。项目中通过 USART2 DMA 空闲中断接收 BT24 蓝牙数据，利用 FreeRTOS 多任务与消息队列完成蓝牙指令、偏航角和电机状态的数据传递；通过软件 I2C 读取 MPU6050 偏航角，并结合左右轮编码器反馈设计 PID 闭环控制，提升小车直线行驶稳定性与转向角度控制精度。同时加入 IWDG 看门狗任务与 OLED 实时显示功能，提高系统运行可靠性和调试效率。
